# 🎵 FreeMusic Hitmos Downloader

Автоматизированный инструмент для переноса ваших любимых плейлистов со стриминговых платформ в виде полноценных MP3-файлов на ваш компьютер **БЕЗ ЦЕНЗУРЫ**.

---

## ⚡ Описание

Данный проект состоит из двух основных частей:
1. **JavaScript-парсер** (для браузера), который извлекает всю информацию о треках из вашего плейлиста Spotify. Для получения TXT файла с других площадок вы можете использовать другие парсеры или браузерные расширения.
2. **Hitmos Downloader** (Windows-программа), которая берет этот список и автоматически находит, скачивает и приводит в идеальный порядок MP3-файлы с музыкального портала Hitmos. В его версии

Программа автоматически:
- Скачивает песню в формате **MP3**.
- Добавляет **правильные метатеги** (Название, Исполнитель, Альбом).
- Извлекает из Spotify **HQ-обложку (640x640)** и "вшивает" её прямо в аудиофайл.

---

## 🚀 Как пользоваться

### Шаг 1. Получение списка треков из Spotify
Вам нужно вытащить список песен из веб-версии Spotify. Сделать это очень просто:

1. Откройте нужный плейлист в веб-версии [Spotify](https://open.spotify.com/) в браузере (Google Chrome, Edge, Яндекс.Браузер).
2. Откройте инструменты разработчика (клавиша `F12` или `Ctrl + Shift + I`).
3. Перейдите во вкладку **Консоль (Console)**.
4. Откройте файл `parsers/spotify_web_parser.js` из этого репозитория, **скопируйте весь его код** и **вставьте в консоль браузера**.
5. Нажмите `Enter`. 
6. Как только парсер дойдет до низа страницы введите `STOP()` и нажмите `Enter`.
7. Скопируйте полученный текст и сохраните его в TXT-файл в удобном для вас месте.

**Примечание:** если вы используете иной музыкальный сервис или у вас уже есть TXT-файл с нужным содержим сразу приступайте к шагу 2.

### Шаг 2. Скачивание MP3
1. Запустите файл `FreeMusic_HitmosDownloader.exe`.
2. При первом запуске скрипт незаметно подгрузит нужные компоненты браузера для работы, просто подождите пару минут.
3. В открывшемся окне программы:
   - В поле **"Папка для сохранения"** выберите папку, куда будут скачиваться ваши песни.
   - В поле **"Файл с треками"** укажите тот самый `.txt` файл, который вы скачали из Spotify на предыдущем шаге.
4. Нажмите **"Начать скачивание"**.
5. Налейте себе чаю ☕ и наблюдайте, как ваша музыкальная коллекция материализуется на вашем диске!

## ⚠️ Важные примечания

> [!NOTE]
> **Не все треки могут быть найдены**  
> Hitmos — огромный, но не бесконечный портал. Если программа не сможет найти трек, она честно пропустит его и добавит в файл `fails.txt` (в вашей папке с музыкой). Эти несколько песен вам придется найти и скачать вручную. Для поиска подходящей версии трека исползуются два алгоритма. Если по какой-то причине один из алгоритмов не смог найти трек, программа автоматически переходит к использованию второго. Все треки скачанные с помощью второго алгоритма помещаются в `prefails.txt`. Треки в это файле рекомендуется проверить на верность скачивания.

> [!TIP]
> **Умный пропуск скачанного**  
> Если загрузка прервалась (например, пропал интернет), вы можете просто запустить программу заново, скормив ей тот же самый `.txt` файл. Она автоматически **пропустит уже скачанные треки** и продолжит ровно с того места, где остановилась!

## 🛠 Для разработчиков
Если вы хотите запустить проект из исходников:
1. Установите Python 3.10+
2. Создайте виртуальное окружение: `python -m venv .venv` и активируйте его.
3. Установите зависимости: `pip install -r requirements.txt`
4. Инициализируйте браузер: `playwright install chromium`
5. Запустите скрипт: `python hitmos_downloader.py`
